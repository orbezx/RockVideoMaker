# RockVideoMaker
Rockvideo Maker 🎸
Rockvideo Maker is a lightweight, Python-based desktop application designed to create professional text animations instantly. Built using the Manim animation engine, it allows users to transform plain text into cinematic motion graphics with full control over visual styles and export settings.

Key Features
Custom Text Animations: Automatically generates "Write" and "Fade" effects for any text input.

Visual Color Control: Includes built-in color pickers to customize both the text and the background.

Green Screen Ready: Easily set a solid chroma-key background for use in video editing software.

Transparency Support: Export videos with a transparent alpha channel (as .mov) for seamless overlays on other footage.

Pro Rendering Options: * Resolution: Select from 480p up to 4K quality.

Frame Rate: Custom FPS control (e.g., 24, 30, or 60) for smooth motion.

High Performance: Features a "Flush Cache" system to ensure every render is fresh and free of old data errors.

How It Works
The app acts as a visual interface for complex code. When you hit Generate, it writes a custom Python script in the background, executes the Manim engine, and opens your finished video file automatically as soon as it's rendered.
